QraphQL + ReactNative
=====================

QraphQL
-------

depends on : `yarn`
install : `yarn` / `npm i`
run : `yarn start` / `npm start`

Runs on port 8888

react-native
------------

`cd client`

depends on : `react-native` and a working android emulator/phone
install : `yarn` / `npm i`

You will need to change the endpoint ip in `index.android.js`

run : `react-native run-android`
